# Objectos Literales
Es un objecto que contiene entre llaves propiedades con formato nombre:valor separados por comas.

```javascript
const persona = {
  manos: 2,
  piernas: 2,
  caminar: () => { console.log('Caminando'); },
  hablar: () => { console.log('Hablando'); },
};

console.log(persona.manos); // 2
```
