### Closure
EN: When a function uses a variable which was declared outside that function. <br />
SP: Cuando una función utiliza una variable la cual fue declarada fuera de dicha función.

```javascript
var name = 'Jhon';

function fullName(){
  name = name + ' Doe';
  return name;
}
```
